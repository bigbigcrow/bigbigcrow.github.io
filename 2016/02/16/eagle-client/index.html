发布eagle-client程序包

NPM（Node Package Manager）称为Node包管理器，主要功能：安转、卸载、更新、查看、搜索、发布…

在发布程序包之前，可以先在本地打包安装一次，测试程序运作是否正常，安装命令为：

npm install <本地包路径>

发布程序包只需要简单的两步：

1、注册NPM账号

要把程序包发布到npmjs.org，还需要先注册一个帐号。npm并没有提供网页版的注册向导，注册也要通过命令行来进行。

npm adduser

执行此命令后，会依次出现输入Username、Password、Email的提示，输入好之后等待一会儿就可以了。

2、发布程序包

npm puslish <本地包路径>

至此，eagle-client程序包就被发布到了npmjs.org上，so easy~

  • 如果要 更新程序包 ，只要修改一下package.json中的版本号，再重新执行发布命令就可以了。

  • 账号注册成功后，npm会把认证信息存储在~/.npmrc文件中，并且可以通过以下命令查看npm当前使用的用户：

    npm whoami

  • npm社区版本号规则采用的是semver(语义化版本)，主要规则如下：
    版本格式：主版号.次版号.修订号。版号递增规则如下：

    主版号：当你做了不相容的 API 修改，
次版号：当你做了向下相容的功能性新增，
修订号：当你做了向下相容的问题修正。
先行版号及版本编译资讯可以加到「主版号.次版号.修订号」的后面，作为延伸。

下面看看如何安装刚刚发布的eagle-client程序包吧~

安装eagle-client程序包

安装eagle-client程序包的方法同其他程序包的安装步骤一致，执行：

>
npm install -g eagle-client
